# ✊✋✌️ Rock Paper Scissors

A fun and interactive **Rock Paper Scissors** game built using **HTML**, **CSS**, and **JavaScript**. Players compete against the computer, which generates random moves. The game keeps track of scores and provides an engaging user experience with a clean and responsive interface.

## 🚀 Features

- Play against the computer
- Random computer move generation
- Real-time score tracking
- Score persistence using LocalStorage
- Responsive and user-friendly interface
- Reset score functionality
- Interactive game animations and UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## 📚 Concepts Used

- DOM Manipulation
- Event Handling
- Conditional Statements
- Functions
- Objects
- LocalStorage
- Template Literals

1. Clone this repository.
